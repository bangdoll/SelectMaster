# SelectMaster - 隨機抽籤輪盤

一個美觀且功能完整的網頁隨機抽籤輪盤工具。

## 功能特色

- 🎯 **隨機抽選** - 公平的隨機選擇機制
- 🎨 **精美設計** - 深色主題，參考 Wheel of Names 設計
- 🔊 **音效系統** - 轉動音效與中獎歡呼聲
- 🎉 **彩花效果** - 中獎時的華麗慶祝動畫
- 📱 **響應式設計** - 支援桌面、平板、手機
- 🗑️ **移除中獎者** - 可選擇刪除已中獎的名單
- 🔇 **靜音控制** - 可開關音效
- ⛶ **全螢幕模式** - 支援全螢幕顯示

## 技術棧

- **Vite** - 快速的建置工具
- **Vanilla JavaScript** - 純 JavaScript，無框架依賴
- **Canvas API** - 用於繪製輪盤
- **Web Audio API** - 音效處理
- **canvas-confetti** - 彩花效果

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 使用方式

1. 在右側面板輸入名單（每行一個名字）
2. 點擊「開始轉動」按鈕或點擊中心圓圈
3. 等待輪盤停止，查看中獎者
4. 可選擇保留或刪除中獎者
