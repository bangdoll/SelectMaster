# Changelog

所有顯著的變更都將記錄在此文件中。

格式基於 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.0.0/)。



## [1.2.0] - 2026-01-18

### 新增
- 🎮 音效切換功能 - 點擊頂部的音符按鈕可切換「機械聲 🎵」與「清脆聲 🎮」
- 🔊 新增「清脆聲」音效選項，提供類似遊戲的輕快回饋音

### 改進
- 🔊 再次調整轉盤音效 - 改用方波 (Square Wave) 搭配低通濾波器 (Lowpass) 掃頻，模擬更紮實的物理/塑膠撞擊聲。

### 新增
- 支援單一選項旋轉 - 即使只剩一個名字，輪盤也能正常轉動

### 修正
- 修正只有一個選項時無法旋轉的限制

## [1.0.0] - 2025-11-26

### 新增
- 🎯 隨機抽籤輪盤核心功能
- 🎨 深色主題設計，參考 Wheel of Names 風格
- 🔊 音效系統 - 轉動音效與中獎歡呼聲
- 🎉 彩花慶祝動畫效果 (canvas-confetti)
- 📱 響應式左右面板佈局
- 🗑️ 移除中獎者功能
- 🔇 靜音控制按鈕
- ⛶ 全螢幕模式
- 👆 點擊中心圓啟動轉動
- 📋 隱藏/顯示側邊面板切換
- 🌐 繁體中文介面本地化

### 技術特性
- 使用 Vite 建置工具
- Canvas API 繪製輪盤
- Web Audio API 音效處理
- 純 JavaScript 無框架依賴

### UI/UX 改進
- 名字正確顯示於扇形區塊
- 文字沿曲線渲染，避免重疊
- 適當的文字間距與大小調整
- 輪盤顏色配置最佳化
- 中心圓縮小 30% 以增加轉動區域
